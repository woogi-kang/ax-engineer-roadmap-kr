# 행동강령

이 프로젝트는 서로 다른 경력, 산업, 기술 선택을 존중하는 협업 공간을 지향한다.

## 기대하는 행동

- 주장과 반론을 사람보다 근거에 연결한다.
- 확인한 사실, 경험에서 나온 관찰, 아직 검증하지 않은 가설을 구분한다.
- 초보적인 질문을 낮춰 보지 않고 필요한 맥락을 설명한다.
- 특정 기술이나 회사 경험을 보편적인 정답으로 강요하지 않는다.
- 사례를 공유할 때 조직·고객·동료의 비공개 정보를 제거한다.
- 잘못을 발견하면 책임 추궁보다 수정 가능한 근거를 먼저 제시한다.

## 허용하지 않는 행동

- 괴롭힘, 차별, 모욕, 신상 공개
- 비공개 회사 정보·개인정보·인증정보 게시
- 확인하지 않은 성과나 경력을 사실처럼 제시
- 광고, 채용 홍보, 유료 서비스 판매를 위한 반복 게시
- 출처를 숨긴 자료 복제

## 신고

공개 Issue에 적기 어려운 문제는 저장소 소유자의 GitHub 프로필에 공개된 연락 수단으로 알려준다. 관리자는 필요한 경우 콘텐츠 삭제, 참여 제한, 공개 범위 조정 등의 조치를 취한다.

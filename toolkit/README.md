# 실무 템플릿

템플릿은 문서를 채우기 위한 형식이 아니라 빠뜨리기 쉬운 판단과 증거를 확인하기 위한 도구다.

## 사용 원칙

- 업무 위험과 규모에 맞게 줄여 쓴다.
- 모르는 값은 추정으로 채우지 않고 `UNKNOWN`으로 둔다.
- 같은 정보를 여러 문서에 복사하기보다 원본 링크를 연결한다.
- 승인과 책임을 “사람”으로만 적지 않고 역할이나 담당자를 지정한다.
- 결과가 기대와 다르면 템플릿보다 실제 결정 기록을 우선한다.
